Agnostic Table

A 2.5kb* client-side view-framework-agnostic table abstraction for Javascript

* when g-zipped. 4kb otherwise.

Features

  • Pagination
  • Filtering
    • Filter by one or many fields
  • Ordering
    • Forward and reverse sorting
